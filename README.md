
# iPhone Sales Data Analysis 📱

This project is a Python-based data analysis notebook that performs Exploratory Data Analysis (EDA) on iPhone sales data from Flipkart (India). Using data science libraries, it uncovers patterns in how Apple products are priced, discounted, and rated by customers in the Indian market.

---

### 📊 Dataset Overview
The dataset contains structured details of **62 Apple products** sold on Flipkart with key attributes:
* **Product Name**: Model name, color, and storage capacity of the iPhone.
* **Sale Price**: The actual price at which the phone is sold.
* **Mrp**: Maximum Retail Price (Original retail price).
* **Discount Percentage**: The markdown percentage offered on the phone[cite: 4].
* **Star Rating**: Average user rating out of 5 stars[cite: 4].
* **Number Of Ratings & Reviews**: Total customer engagement metrics[cite: 4].
* **RAM**: System memory of the device[cite: 4].

---

### 🔍 Key Features of the Analysis
1. **Data Cleaning**: Checked for null values (found **0 null values**) and duplicates (found **0 duplicate rows**) to ensure data integrity[cite: 4].
2. **Statistical Summary**: Analyzed average price points (~₹80,073), average discounts (~10%), and customer sentiment[cite: 4].
3. **Product Ranking**: Extracted the Top 10 highest-rated iPhones on Flipkart to find customer favorites (dominated by the iPhone 11 Pro Max at 4.7 stars and iPhone XR/8 Plus at 4.6 stars)[cite: 4].
